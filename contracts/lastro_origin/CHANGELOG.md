# Changelog

Changelog for `lastro_contracts`.

## [0.1.0] - 2026-06-23

### Added

- `proof_of_origin` module: `ProofOfOrigin` contract for registering reference
  seals and recording origin attestations on-chain.
- `mint_gate` module: symbolic mint gate that consumes valid origin proof.

### Removed

- Example `flipper` module, replaced by Lastro origin-proof contracts.
