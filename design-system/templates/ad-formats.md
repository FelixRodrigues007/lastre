# Lastro Ad Formats

Use these specs as the default crop/layout set for Lastro advertising materials.

## Core formats

| Format | Size | Use | Layout recipe |
|---|---:|---|---|
| LinkedIn feed | 1200×627 | B2B launch/ad posts | Left headline, right proof rail, bottom mark |
| Social square | 1080×1080 | LinkedIn/X/Meta | Centered headline, diagram below, proof chip footer |
| Story/Reel cover | 1080×1920 | Vertical story, event teasers | Top badge, large stacked headline, vertical proof rail |
| X wide image | 1600×900 | Technical threads | Wide architecture rail with code/hash detail |
| Demo thumbnail | 1280×720 | YouTube/demo video | High-contrast headline + Valid/Invalid split |
| Display rectangle | 1200×628 | Display/retargeting concept | Short headline, single CTA, minimal detail |
| Medium rectangle | 300×250 | Display test | 3-word headline + mark + CTA only |

## Layout grid

### Feed/wide assets
- Outer margin: 64px.
- Grid: 12 columns, 32px gutter.
- Safe text width: 6 columns.
- Proof rail width: 4 columns.
- Footer height: 64px.

### Square assets
- Outer margin: 72px.
- Headline block: top 45%.
- Proof module: middle 35%.
- Footer: bottom 20%.

### Story assets
- Outer margin: 80px top/bottom, 56px sides.
- Keep critical text between 250px and 1600px vertically.
- CTA sits above platform UI controls, not at the absolute bottom.

## Component placement

1. **Eyebrow badge:** top-left or top-center.
2. **Headline:** largest text, one idea only.
3. **Support line:** one sentence; never more than 18 words.
4. **Proof rail:** always left-to-right or top-to-bottom; never circular.
5. **Footer:** Lastro mark + category, optional `Demo uses fictional data` note.

## Export guidance

- SVG for editable templates.
- PNG for social upload.
- Keep a source file with text editable.
- Do not rasterize small hash text unless required.
- Validate at 50% zoom to simulate feed compression.
