# Support

Lastre is currently a demonstration/prototype proof-of-provenance stack.

## Public demo support

If the frontend is unavailable:

1. Check `https://lastre.io` in an incognito window.
2. Check `https://app-api.lastre.io/api/health`.
3. Review `docs/DEPLOYMENT_RUNBOOK.md`.

## Developer support

Start with:

- `docs/README.md`
- `docs/API_CONTRACT.md`
- `docs/FRONTEND_ROUTES.md`
- `docs/DEPLOYMENT_RUNBOOK.md`

## What Lastre is not

Lastre is not an investment product, yield product, token sale, ownership claim,
or trading marketplace. The demo uses fictional assets only.
